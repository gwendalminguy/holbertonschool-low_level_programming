# Low-Level Programming

This repository focuses on low-level programming languages.

* [C - Hello World](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/hello_world), a project to learn about the C syntax and using a compiler.

* [C - Variables, If/Else & Loops](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/variables_if_else_while), a project to learn about using variables, conditionals and loops.

* [C - Functions & Nested Loops](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/functions_nested_loops), a project to learn about using functions and nested loops.

* [C - More Functions & Nested Loops](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/more_functions_nested_loops), a project to learn about using more functions and nested loops.

* [C - Pointers, Arrays & Strings](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/pointers_arrays_strings), a project to learn about using pointers, and manipulating arrays and strings.

* [C - Recursion](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/recursion), a project to learn about using recursion.

* [C - Argc & Argv](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/argc_argv), a project to learn about using command-line arguments.

* [C - Malloc & Free](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/malloc_free), a project to learn about allocating and freeing memory dynamically.

* [C - More Malloc & Free](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/more_malloc_free), a project to learn more about allocating and freeing memory dynamically.

* [C - Structures & Typedef](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/structures_typedef), a project to learn about using structures and typedef.

* [C - Function Pointers](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/functions_pointers), a project to learn about using function pointers.

* [C - Variadic Functions](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/variadic_functions), a project to learn about using functions with a variable number of arguments.

* [C - Singly Linked Lists](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/singly_linked_lists), a project to learn about manipulating singly linked lists and nodes.

* [C - Doubly Linked Lists](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/doubly_linked_lists), a project to learn about manipulating doubly linked lists and nodes.

* [C - Bit Manipulation](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/bit_manipulation), a project to learn about manipulating bits by using bitwise operators.

* [C - File I/O](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/file_io), a project to learn about opening, reading and writing files.

* [C - Numbers](https://github.com/gwendalminguy/holbertonschool-low_level_programming/tree/main/numbers), a project to play with numbers properties.
